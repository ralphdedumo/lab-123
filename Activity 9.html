Activity 9 ( Grid Layout Design 4x4 )

Working in Illustrator, InDesign, Photoshop, or another page layout program, draw 16 squares on a page (4 rows, 4 columns).

1. Divide each square into nine more squares to create 4 rows by 4 column box.

2. Fill in the sixteen square with color to create different compositions within the same structure. Imagine each gridded box as a web page.

Legend:
Dark-Colored-Tone boxes = images
Light-Colored-Tone boxes = contents/texts

3. Repeat the exercise, using the 1–square grid to layout boxes representing text and image.

4. Create different variation for each page.
